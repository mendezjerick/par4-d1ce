Course Requirement Map

Visible in the UI
Process Management: Process table with PID, burst time, memory need, waiting time, turnaround time, and states.
CPU Scheduling: Round Robin controls, execution order, averages, and Gantt chart timeline.
Memory Management: Fixed partitions, waiting list, allocation, and release behavior.
File System: Create, delete, list, and display file contents using in-memory storage.
I/O Spooling: Add jobs to a printer queue and process them one at a time.
